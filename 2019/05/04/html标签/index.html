html标签

html(超文本标记语言——HyperText m=Markup Language):

html使用”标记”来注明文本、图片和其他内，由一系列元素组成。

一个元素由开始标签、内容、结束标签构成。

嵌套元素：将元素放到其他元素里称作嵌套。

html的两类元素类别：块级元素和内联元素。

空元素只有一个标签，如 img（插入图片). alt 属性可以在图片不能显示时对图片文字描述。

元素属性：1在元素和属性之间有个空格(如果有一个或多个已存在的属性，就与前一个属性之间有一个空格）。

​ 2属性后面要跟着一个“=”符号。

​ 3属性值由引号引起来。

a 锚元素使被包裹的内容变成超链接，锚元素可以添加大量属性，如href声明超链接地址，titie声明额外信息。(HTTP GET请求)

与 a 标签相对应的 form 标签可以进行HTTP POST请求，当然也可以进行GET请求。

head 包含一些不在页面显示的元数据。meta 专门用来添加元数据。

input 默认type是text，其有多个type，举例如下：

  1. button
  2. checkbox：复选框，必须用value属性定义提交的值。可以多选

  3. radio：与chckbox对应，必须用value属性定义提交的值，但只能单选。

    button 可以添加内容，这是其与input的不同之处，其type和input有不少相同。

table:用于建立表格，td之间会平行建立，用tr包裹td建立一行。
