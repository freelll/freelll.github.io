css的布局主要分成两类：左右布局和上下布局。

布局的主要方式来自float和position。

float（浮动）：

  • left：将元素浮动到左侧。
  • right：将元素浮动到右侧。
  • none：不浮动。
  • inherit：继承。

position：

  • relative（相对定位）：不脱离文档流的移动。
  • absolute（绝对定位）：脱离文档流的移动。

float与absolute脱离文档流的区别是：float会产生文字环绕效果，absolute则完全覆盖文字。

块级元素并排的方法：

在子元素上设置float，在父元素上设置. clearfix 消除浮动，设置子元素的width可以改变并排的布局。

左右布局：

左侧定宽，右侧自适应：

  • float+margin
  • absolute+margin

左中右布局：

  • 左右两侧float，中间自适应。
  • 左右两侧absolute定位，中间自适应。

水平居中的方法：

  • 行内元素：text-align：center(行内元素，其父元素是块级元素直接给父元素设置text-align：center)，其父元素不是块级元素则先将父元素变成块级元素再text-align：cener。
  • 块级元素：margin：0 auto（已设置width），未设置width的情况下且父元素是块级元素，将子元素display：inline-block，然后text-align：center。

水平居中的方法：

  • 单行的行内元素：line-height=height即可。
  • 多行的行内元素：给父元素设置display：table-cell和vertical-align：middle。
  • 块级元素：父元素相对定位然后子元素绝对定位，设置子元素的top：50%。定高度的情况下可以

直接设置transform：translateY（-50%）；不定高度的情况下课直接设置transform：translateY（-50%）。
